# How to use

## Layout

- `frontend/` - UI
- `backend/` - API/services
- `infra/` - IaC templates (Terraform etc.)

## Run locally

### Backend

```bash
cd backend
# see backend/api for specifics
```

### Frontend

```bash
cd frontend
# see frontend/ui for specifics
```

## Deploy

```bash
cd infra
# see infra/ccaas-template
```
